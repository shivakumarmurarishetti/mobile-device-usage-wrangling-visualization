# A Data-Driven Insight into Mobile Device Usage and User Behaviour

This project was completed as part of the **Applied Data Wrangling & Visualization** module for the **MSc in Information Systems** at **University College Dublin (UCD)**.  
The goal of the project is to clean, analyze, and visualize mobile device usage data to uncover meaningful patterns in user behaviour.
